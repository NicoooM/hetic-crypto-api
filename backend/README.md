# backend

> Projet initialisé avec [Bun](https://bun.sh)

# Lancer le projet

Installer les dépendances :

```bash
bun i
```

Lancer le serveur :

```bash
bun dev
```

Générer le client Prisma :

```bash
bunx prisma generate
```

# Développer le projet

Pour créer une nouvelle migration :

```bash
bunx prisma migrate dev
```
